## 테스트 체크리스트

### 사용자 플로우 테스트

#### 신규 사용자 플로우
```
1. 랜딩 페이지 접속
2. "무료로 시작하기" 클릭
3. 온보딩 5단계 완료
   - 산업 선택
   - 창업 단계 선택
   - 비즈니스 유형 선택
   - 고민 선택 (복수)
   - 목표 선택
4. 커리큘럼 생성 로딩
5. 커리큘럼 결과 확인
6. 첫 콘텐츠 시청
7. 회원가입/로그인 유도
```

#### 로그인 사용자 플로우
```
1. 로그인
2. 대시보드 확인
3. 이어서 학습하기
4. 콘텐츠 시청
5. 진행률 업데이트 확인
6. AI 챗봇 사용
7. 새 커리큘럼 생성
```

## 피드백 수집 항목

### 정량적 지표

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 온보딩 완료율 | 70%+ | GA 퍼널 |
| 커리큘럼 생성율 | 80%+ | API 로그 |
| 첫 시청 전환율 | 60%+ | 이벤트 트래킹 |
| 일일 재방문율 | 30%+ | 코호트 분석 |
| AI 챗봇 사용률 | 20%+ | 이벤트 트래킹 |

### 정성적 피드백

#### 온보딩 관련
- 질문이 이해하기 쉬웠나요?
- 선택지가 충분했나요?
- 온보딩이 너무 길거나 짧았나요?

#### 커리큘럼 관련
- 추천된 커리큘럼이 만족스러웠나요?
- AI 추천 이유가 납득됐나요?
- 콘텐츠 구성이 체계적이었나요?

#### 콘텐츠 관련
- 영상 품질은 만족스러웠나요?
- 원하는 내용을 찾을 수 있었나요?
- 콘텐츠 길이가 적절했나요?

#### AI 챗봇 관련
- AI 멘토가 도움이 됐나요?
- 답변이 이해하기 쉬웠나요?
- 개인화된 느낌을 받았나요?

## 베타 테스터 운영

### 모집 대상
- 예비 창업자 ~ Pre-Seed 초기 창업자
- IT/테크, 커머스, 서비스업 관심자
